## ✨ 功能

### 📊 定时爬取

- **立即爬取**：马上执行一次
- **每天定时**：每天指定时间自动爬取
- **每周定时**：每周指定时间自动爬取
- **自定义时间**：灵活设置爬取周期

### 📈 数据统计

爬取完成后自动显示：

- 每个用户的新增视频数
- 每个用户的新增评论数
- 关注数增幅
- 粉丝数增幅

### 🖼️ 图文转视频

- 自动检测抖音图文作品（图片+音频格式）
- 使用 ffmpeg 将图片和音频合并为视频文件
- 需要安装 ffmpeg 才能使用此功能

### 💾 数据存储

按用户自动组织数据：

```
data/douyin/
├── 用户A_抖音号/
│   ├── 001_视频标题/
│   │   ├── media/video.mp4
│   │   ├── desc.md
│   │   └── comments.md
│   └── README.md
├── 用户B_抖音号/
│   └── ...
├── crawl_log.md        # 爬取日志
└── history_data.json   # 历史数据（用于对比变化）
```

## 🚀 快速开始

### 1. 安装依赖

```shell
uv sync
```

### 2. 安装 ffmpeg（可选，图文转视频需要）

如果需要将抖音图文作品转换为视频，需要安装 ffmpeg：

- **下载地址**：https://ffmpeg.org/download.html
- **Windows**：下载后解压，将 `bin` 目录添加到系统 PATH 环境变量
- **验证安装**：终端输入 `ffmpeg -version`，正常显示版本号即可

> 💡 如果不安装 ffmpeg，图文作品只会下载图片和音频，不会自动合并为视频。

### 3. 配置目标用户

编辑 `schedule_crawler.py`，修改 `TARGET_USERS`：

```python
TARGET_USERS = [
    "https://www.douyin.com/user/用户ID1?from_tab_name=main",
    "https://www.douyin.com/user/用户ID2?from_tab_name=main",
]
```

### 4. 配置 Chrome 浏览器

1. 安装 Chrome 浏览器（版本 >= 144）
2. 地址栏输入 `chrome://inspect/#remote-debugging`
3. 勾选 "Allow remote debugging for this browser instance"

### 5. 运行

```shell
uv run python schedule_crawler.py
```

选择模式：

```
1. 立即爬取
2. 每天 00:00 爬取
3. 每周一 00:00 爬取
4. 自定义时间
```

## ⚙️ 配置说明

编辑 `config/base_config.py`：

| 配置项                | 说明                         | 默认值 |
| --------------------- | ---------------------------- | ------ |
| ENABLE_CDP_MODE       | 连接已有 Chrome 浏览器       | True   |
| ENABLE_GET_COMMENTS   | 爬取评论                     | True   |
| ENABLE_GET_MEIDAS     | 下载媒体资源（视频/图片）    | True   |
| ENABLE_DEDUP          | 数据去重（跳过已爬取的视频） | True   |
| CRAWLER_MAX_SLEEP_SEC | 请求间隔（秒）               | 8      |
